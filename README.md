# TextSummarize

A text summarization project using extractive and abstractive methods.

## Features
- Extractive summarization with Sumy
- Abstractive summarization with Transformers
- ROUGE evaluation metrics
- Streamlit web interface
- Jupyter notebook examples

## Installation
```bash
pip install -r requirements.txt
```

## Usage

### Command Line
```bash
python main.py
```

### Web Interface
```bash
streamlit run streamlit.py
```

### Jupyter Notebook
```bash
jupyter notebook summarize.ipynb
```

## Dependencies
- sumy
- transformers
- datasets
- rouge-score
- streamlit
- nltk
